Esta app se hizo con flutter utilizando estos paquetes: - Riverpod (gestor de estado) - arquitectura limpia (estructura del aplicativo) - SQLFlite (datos internos) - Shared Preferences (almacenamiento dentro del dispositivo)

Proyecto de aplicativo móvil

InfoMovil: comparar teléfonos - Proyecto personal 18/11/2024 - En curso

Este proyecto personal tiene el objetivo de ayudar al usuario en conocer las caracteristicas que tiene los celulares recien salidos al mercado. También, podrán hacer una comparación entre dos telefonos e ingresar al chatGPT enviando una pregunta predefinida sobre la diferencia de los dos celulares seleccionados. Esta app se hizo con flutter utilizando estos paquetes: - Riverpod (gestor de estado) - arquitectura limpia (estructura del aplicativo) - SQLFlite (datos internos) - Shared Preferences (almacenamiento dentro del dispositivo) - HTTP (conexión API) Este aplicativo esta pasando la fase de prueba cerrada requerida por play store.
